1. システム概要

このアプリケーションは、マーケティング教育動画のAIタグ生成システムです。大量の動画データに対して自動的にタグを生成し、検索性とカテゴライゼーションを向上させます。

主要機能

  • Google Sheetsからの動画データ自動読み込み
  • 高度なAIタグ生成アルゴリズム（10-15個/動画）
  • バッチ処理による大量データ処理
  • WAF対策によるエラー回避システム
  • CSV・JSON形式での結果出力
  • リアルタイム進捗監視と処理制御

2. 機能一覧

優先度: 高

詳細:

  • クライアントサイド完全処理への移行

  • 100%エラー回避の実現

F002: 高度AIタグ生成システム 完了

概要: 1件あたり10-15個の詳細で重複の少ないタグを自動生成

優先度: 高

詳細:

  • 数値パターン認識（6要素、8分類等）
  • 重複削減アルゴリズム（約40%削減）
  • 動画内容の高精度識別

4. データモデル

主要エンティティ

VideoData (動画データ)

{
    "title": "string",
    "description": "string",
    "summary": "string",
    "skill": "string",
    "level": "string",
    "tags": ["string"],
    "generatedAt": "datetime"
}

BatchResult (バッチ処理結果)

{
    "batchNumber": number,
    "totalItems": number,
    "processedItems": number,
    "processingMode": "single | continuous",
    "strategy": "default | creative | technical",
    "startTime": "datetime",
    "endTime": "datetime",
    "results": [VideoData]
}

TagGenerationConfig (タグ生成設定)

{
    "strategy": "default | creative | technical",
    "tagCount": {"min": 10, "max": 15},
    "duplicationThreshold": 0.4,
    "categories": ["skill", "content", "level", "industry"],
    "exclusionPatterns": ["string"]
}
