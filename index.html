✨ 主要機能

  • 自動ドキュメント生成 - チャット内容から自動的にドキュメントを生成・更新
  • 進捗可視化 - プロジェクトの進捗をグラフィカルに表示
  • チャット自動保存 - 2時間ごとにチャット履歴を自動バックアップ
  • テスト管理 - テストケースの管理と実行結果の記録

🚀 セットアップ方法

  1. リポジトリをクローン（Gitを使用する場合）
  2. Python 3.8以上をインストール
  3. 必要なパッケージをインストール: pip install schedule
  4. 自動更新スクリプトを実行: python docs/auto_updater.py
  5. チャット保存スクリプトを実行: python scripts/chat_backup.py

📝 スクリプト実行方法

ドキュメント自動更新

# 監視モードで実行（推奨）
python docs/auto_updater.py

# 1回だけ実行
python docs/auto_updater.py --once

チャット自動保存

# デフォルト設定（2時間ごと）で実行
python scripts/chat_backup.py

# バックアップ一覧を表示
python scripts/chat_backup.py --list
